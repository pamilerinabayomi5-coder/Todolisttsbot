# 📋 TodoListTSBot

A production-ready Telegram Bot built with Python 3.12+ and `python-telegram-bot` (v21+). Designed for tasks organization, flexible user timezones, priority tagging, categorization matrix options, custom recurring reminders, and bulk exports.

## 🚀 Features
- **Conversational Wizardry:** Guided multi-step creation pathways via `ConversationHandler`.
- **Dynamic Scheduler:** Validates and triggers task alert updates accurately.
- **Data Customization:** Supports recurring task adjustments (Daily, Weekly, Monthly) and manual timezone specification.
- **Data Portability:** Instantly backup/restore tasks using standard CSV structures.
